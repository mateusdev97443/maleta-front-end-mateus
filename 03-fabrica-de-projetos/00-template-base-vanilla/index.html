Base front-end sem frameworks

Template limpo para projetos HTML, CSS e JavaScript.

Use esta base para começar projetos pequenos com semântica, organização, responsividade e critérios de qualidade desde o primeiro commit.

Ver briefing

Objetivo

Construir uma fundação reutilizável.

O template separa responsabilidades para facilitar manutenção, estudo e evolução gradual sem adicionar ferramentas antes da hora.

Estrutura

Arquivos organizados por responsabilidade.

HTML semântico

Header, navegação, conteúdo principal, seções nomeadas e rodapé.

CSS modular

Reset, variáveis, base, layout, componentes e responsividade.

JavaScript vanilla

Interações progressivas sem bibliotecas externas.

Qualidade

Antes de entregar, revise o essencial.

  • Layout testado em telas pequenas e grandes.
  • Links, foco e navegação funcionando pelo teclado.
  • Conteúdo claro, sem textos provisórios escondidos.
  • CSS e JavaScript organizados por intenção.
