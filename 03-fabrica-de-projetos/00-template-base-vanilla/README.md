# Template Base Vanilla

Projeto inicial para criar páginas front-end sem frameworks e sem bibliotecas externas.

## Objetivo

Fornecer uma fundação limpa para praticar HTML semântico, CSS organizado, layout mobile-first e JavaScript progressivo.

## Estrutura

```text
00-template-base-vanilla/
├── index.html
├── assets/
│   ├── css/
│   │   ├── reset.css
│   │   ├── variables.css
│   │   ├── base.css
│   │   ├── layout.css
│   │   ├── components.css
│   │   └── responsive.css
│   └── js/
│       └── main.js
└── docs/
    ├── briefing.md
    └── checklist-qualidade.md
```

## Como executar

Abra o arquivo `index.html` diretamente no navegador ou use uma extensão de servidor local do editor.

## Aprendizados esperados

- Criar estrutura HTML semântica.
- Separar CSS por responsabilidade.
- Aplicar mobile-first.
- Implementar interações simples com JavaScript vanilla.
- Revisar acessibilidade e qualidade antes da entrega.

## Checklist de entrega

- [ ] Conteúdo principal compreensível sem CSS.
- [ ] Layout funcional em telas pequenas.
- [ ] Navegação por teclado preservada.
- [ ] Estados de foco visíveis.
- [ ] CSS separado por responsabilidade.
- [ ] JavaScript sem dependências externas.
- [ ] Documentação mínima atualizada.
