# Fábrica de Projetos

Área para criar, repetir e evoluir projetos front-end com escopo claro, critérios de qualidade e base escalável.

## Fluxo recomendado

1. Escolher uma categoria.
2. Definir briefing e objetivo de aprendizado.
3. Copiar o template base vanilla quando fizer sentido.
4. Implementar em ciclos pequenos.
5. Revisar com checklist de qualidade.
6. Registrar aprendizados no dossiê de evolução.
