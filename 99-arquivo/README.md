# Arquivo

## Objetivo

Guardar conteúdos antigos, pausados, substituídos ou fora do escopo ativo sem apagar histórico e sem poluir a estrutura principal da Maleta.

## Quando usar

Use esta área quando um material não deve mais orientar estudo ou implementação atual, mas ainda precisa ser preservado por histórico, comparação ou retomada futura.

## Como organizar

- Mover apenas conteúdos realmente obsoletos, pausados ou substituídos.
- Informar motivo do arquivamento quando necessário.
- Evitar misturar arquivo com backlog ativo.
- Não usar esta pasta como depósito de arquivos sem revisão.

## Status atual

**Em evolução.** A área existe para preservar histórico com controle e evitar que a base principal acumule ruído.

## O que já existe

- README com objetivo e regras iniciais.
- Espaço reservado para materiais arquivados quando surgirem.

## O que será evoluído depois

- Critérios mais detalhados de arquivamento.
- Modelo simples para registrar motivo de arquivo.
- Organização por data, fase ou tipo de material, se houver volume suficiente.

## Regra para não poluir a Maleta

Não mova para o arquivo algo que ainda precisa ser finalizado. Backlog, rascunho ativo e material obsoleto são estados diferentes e devem ser comunicados com clareza.
