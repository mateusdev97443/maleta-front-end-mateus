# Exercícios Práticos

Resolva em arquivos simples. Não copie soluções prontas: o objetivo é treinar leitura do DOM, decisão e teste manual.

## Exercício 1: Selecionar elementos

**Objetivo:** praticar selecionar elementos.

**Enunciado:** crie uma pequena interface relacionada ao tema e faça uma interação com JavaScript puro.

**Requisitos:** use classes claras, selecione os elementos necessários, registre eventos quando fizer sentido e mostre feedback visível ao usuário.

**Dica:** comece com HTML mínimo, teste o seletor no console e só depois escreva a alteração.

**Critério de conclusão:** a interação funciona sem erro no console e o código pode ser explicado em voz alta.
