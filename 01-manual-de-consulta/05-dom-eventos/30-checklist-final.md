# Checklist Final

Use esta lista antes de considerar uma interação pronta.

## Estrutura

- A pasta da fase existe.
- Os 35 arquivos existem e possuem conteúdo.
- Cada arquivo tem apenas um H1.
- O README aponta para todos os capítulos.

## Código DOM

- Elementos são selecionados com nomes claros.
- Há verificação de existência antes de manipular.
- Eventos usam `addEventListener`.
- Formulários usam `preventDefault` quando a página não deve recarregar.
- Mudanças visuais usam classes CSS sempre que possível.
- `innerHTML` só aparece com alerta de cuidado.

## Qualidade

- O conteúdo está em português do Brasil.
- Os exemplos são pequenos, completos e progressivos.
- Os exercícios não entregam tudo resolvido.
- Os mini desafios não usam comunicação externa nem persistência.
- Não houve merge na `main`.
