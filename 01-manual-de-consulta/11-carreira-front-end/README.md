# Carreira Front-end

## Objetivo

Organizar evolução profissional, portfólio, comunicação e rotina de estudos.

## Quando usar

Use este guia ao planejar estudos, preparar projetos, revisar currículo e praticar entrevistas.

## Conceitos principais

- Fundamentos essenciais do tema.
- Boas práticas que devem virar padrão nos projetos.
- Critérios de decisão para escolher a solução mais simples e correta.
- Relação do tema com qualidade, manutenção e experiência do usuário.

## Erros comuns

- Copiar soluções sem entender o motivo.
- Misturar responsabilidades entre estrutura, estilo e comportamento.
- Ignorar nomes claros, organização e consistência.
- Deixar validações e revisões apenas para o fim do projeto.

## Checklist

- [ ] Entendi o problema antes de aplicar a solução.
- [ ] Usei nomes claros e consistentes.
- [ ] Mantive o código simples e organizado.
- [ ] Revisei acessibilidade, responsividade e manutenção.
- [ ] Registrei exemplos úteis para consultas futuras.
