# 16 — Mini desafios HTML

## Desafio 1 — Página de perfil

Briefing: criar uma página pessoal simples para apresentação profissional.

Requisitos: `header`, `main`, `section`, lista de habilidades, links sociais e `footer`.

Critérios: headings corretos, links descritivos e estrutura semântica.

## Desafio 2 — Card de notícia

Briefing: criar um card que possa existir isoladamente em uma lista de notícias.

Requisitos: `article`, título, resumo, data, link para leitura e imagem com `alt`.

Critérios: usar `article` corretamente e evitar conteúdo genérico.

## Desafio 3 — Formulário de inscrição

Briefing: criar formulário para inscrição em evento.

Requisitos: nome, e-mail, telefone, seleção de turno, aceite de termos e botão de envio.

Critérios: labels conectados, `required`, tipos de input adequados e agrupamento lógico.

## Desafio 4 — Tabela de agenda

Briefing: apresentar programação de um workshop.

Requisitos: `caption`, cabeçalho, corpo, horários, temas e responsáveis.

Critérios: `scope` correto e tabela usada apenas para dados tabulares.
