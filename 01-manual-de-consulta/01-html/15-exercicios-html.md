# 15 — Exercícios práticos de HTML

## Básico

1. Crie uma página com estrutura completa: `doctype`, `html lang`, `head`, `body`, `charset`, `viewport`, `title` e `description`.
2. Monte um artigo com `h1`, `h2`, parágrafos, `strong`, `em`, `blockquote` e `code`.
3. Crie uma lista de estudos com `ul`, `ol` e `dl`.

## Intermediário

1. Crie uma página institucional com `header`, `nav`, `main`, três `section` e `footer`.
2. Monte uma tabela de comparação com `caption`, `thead`, `tbody`, `tfoot`, `th`, `td` e `scope`.
3. Crie um formulário de contato com labels, validação HTML e autocomplete.

## Aplicado em projeto real

1. Faça a estrutura HTML de uma landing page para um curso online.
2. Crie uma página de produto com imagem, descrição, benefícios, tabela técnica e formulário de interesse.
3. Audite o HTML de um projeto antigo e liste dez melhorias possíveis.

## Critérios de correção

- Semântica adequada.
- Headings organizados.
- Formulários acessíveis.
- Imagens com `alt` correto.
- HTML legível, indentado e validável.
