# Mini desafios CSS

## Desafio 1 — Card de perfil

**Briefing:** criar um card visual para perfil de pessoa.

**Requisitos:** imagem fluida, nome, descrição, link, borda, sombra sutil e espaçamento consistente.

**Critérios:** CSS externo, classes claras, foco visível no link e sem altura fixa desnecessária.

## Desafio 2 — Página de artigo

**Briefing:** estilizar uma página de leitura.

**Requisitos:** largura máxima confortável, boa tipografia, links identificáveis, espaçamento entre seções e contraste adequado.

**Critérios:** texto legível, `line-height` adequado e hierarquia visual feita com CSS sem quebrar headings.

## Desafio 3 — Formulário simples

**Briefing:** estilizar formulário de contato.

**Requisitos:** labels, campos, botão, estado de foco e mensagem auxiliar.

**Critérios:** não remover outline sem alternativa, usar `font: inherit` nos campos e manter alinhamento básico.

## Desafio 4 — Tabela de dados

**Briefing:** criar estilo para tabela de agenda.

**Requisitos:** cabeçalho destacado, bordas leves, padding e alinhamento à esquerda.

**Critérios:** legibilidade, contraste e ausência de decoração excessiva.

## Desafio 5 — Refatoração

**Briefing:** receber um CSS repetitivo e melhorar sem alterar o HTML além de classes necessárias.

**Requisitos:** criar variáveis, remover repetição e reduzir especificidade.

**Critérios:** resultado visual equivalente, código mais curto e fácil de explicar.
