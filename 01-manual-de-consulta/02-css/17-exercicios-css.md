# Exercícios CSS

## Básico

1. Crie um HTML simples e aplique CSS externo.
2. Estilize `body`, `h1`, `p`, `a` e `button`.
3. Crie classes `.texto-destaque`, `.botao` e `.card`.
4. Diferencie `margin` e `padding` em exemplos visuais.
5. Teste `:hover` e `:focus-visible` em links.

## Intermediário

1. Crie um conjunto de variáveis em `:root` para cores e espaçamentos.
2. Monte três cards com mesma classe base e pequenas variações.
3. Corrija um conflito de especificidade sem usar `!important`.
4. Refaça um trecho com seletores longos usando classes claras.
5. Crie um formulário com estilos de campo, erro textual e foco visível.

## Aplicado em projeto real

1. Pegue uma página HTML existente e crie uma folha de estilos externa.
2. Defina base global, container, botões, cards, links e formulário.
3. Revise contraste, foco, repetição e nomes de classes.
4. Documente três decisões de estilo no próprio CSS ou no README do projeto.
5. Faça uma revisão final usando o checklist deste manual.
