# Mini desafios

Resolva sem DOM e sem interação com tela. Tudo deve funcionar com valores fixos e `console.log`.

## Desafio 1 — Calculadora simples de preço final

### Missão

Construir calculadora simples de preço final usando JavaScript puro.

### Regras

Use dados fixos, funções pequenas e mensagens claras no console.

### O que testar

Teste caso comum, caso limite e caso inválido simples.

### Critério de sucesso

A solução entrega resultado correto e é fácil de ler.
