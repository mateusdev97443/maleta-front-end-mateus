# Checklist final

Use esta lista antes de considerar um exercício ou arquivo JavaScript básico concluído.

- [ ] Variáveis têm nomes descritivos.
- [ ] `const` foi usado por padrão.
- [ ] `let` aparece apenas quando existe reatribuição.
- [ ] Não há uso de `var`.
- [ ] Comparações usam `===` e `!==`.
- [ ] Funções têm responsabilidade clara.
- [ ] Retornos estão corretos e previsíveis.
- [ ] Loops possuem condição de parada.
- [ ] Arrays são percorridos corretamente.
- [ ] Objetos possuem propriedades claras.
- [ ] Não há duplicação desnecessária.
- [ ] `console.log` foi removido ou justificado.
- [ ] Erros comuns foram revisados.
- [ ] Código está legível e bem indentado.
